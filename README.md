## Dependencies

Version of create react app:

```bash
npx create-react-app@3.4.1
```

Core material-ui package

```bash
yarn add @material-ui/core
```

Packages for icons

```bash
yarn add font-awesome@4.7.0
yarn add @material-ui/icons
```

## Start Project

- yarn start

## Languages/Libraries used

- react
- react-router
